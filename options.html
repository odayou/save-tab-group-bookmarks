Bookmark Tab Group

中文 / Chinese

浏览器标签页太多？不常用的标签组舍不得删？用这个插件一键保存到收藏夹吧！

轻松将标签组保存到收藏夹，支持自定义保存位置和右键快速操作

English

Too many tabs in your browser? Don't want to delete unused tab groups? Use this extension to save them to bookmarks with one click!

Easily bookmark tab groups with custom save locations and right-click operations

使用说明 / Usage Instructions

核心功能 / Core Features

1. 保存标签组

  • 中文：选择要保存的标签组，选择收藏夹位置，点击"保存"按钮即可将标签组保存到收藏夹。
  • English：Select the tab groups to save, choose a bookmark location, and click the "Save" button to save the tab groups to bookmarks.

2. 批量保存

  • 中文：可以同时选择多个标签组进行批量保存，提高效率。
  • English：You can select multiple tab groups for batch saving to improve efficiency.

3. 智能重复检测

  • 中文：基于标签内容进行匹配，智能检测重复的标签组，避免重复保存。
  • English：Based on tab content matching, intelligently detects duplicate tab groups to avoid duplicate saves.

4. 自定义保存位置

  • 中文：可以搜索并选择任意收藏夹位置，还可以设置默认保存位置。
  • English：You can search and select any bookmark location, and also set a default save location.

5. 右键菜单操作

  • 中文：在标签页上右键点击，可以快速将当前标签组添加到收藏夹。
  • English：Right-click on a tab to quickly add the current tab group to bookmarks.
